## 🧩 Projects Overview

### 🔐 1. Login & Registration System

A console-based authentication system built in two versions:

#### 🔹 Version 1:

* Uses `vector` for storing user data
* Supports:

  * User Registration
  * Login System
  * Duplicate username validation

#### 🔹 Version 2:

* Uses **File Handling (`fstream`)**
* Stores user credentials in a file
* Features:

  * Persistent data storage
  * Login authentication from file
  * Display stored users

📌 **Concepts Used:**
Vectors, File Handling, Input Validation, String Manipulation

---

### 🧮 2. CGPA Calculator

A system that calculates **GPA and CGPA** based on courses and semesters.

#### ✨ Features:

* Add courses with grades & credit hours
* Calculate total credit hours
* Calculate total grade points
* Compute **GPA (per semester)**
* Compute **CGPA (overall)**
* Multi-semester support

📌 **Concepts Used:**
Classes & Objects, Vectors, Nested Data Structures, Mathematical Logic

---

### 🧩 3. Sudoku Game & Solver

An interactive Sudoku game combined with an automatic solver.

#### 🎮 Game Features:

* User plays Sudoku by entering row, column, and number
* Real-time validation (row, column, 3×3 grid)
* Prevent overwriting filled cells
* Limited attempts (game logic)
* Win detection

#### 🤖 Solver Features:

* Solves Sudoku using **Backtracking Algorithm**
* Recursively fills grid with valid numbers
* Displays complete solution

📌 **Concepts Used:**
2D Arrays, Recursion, Backtracking, Game Logic
