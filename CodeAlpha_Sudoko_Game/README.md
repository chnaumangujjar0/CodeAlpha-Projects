# 🧩 Sudoku Game & Solver (C++)

A console-based **Sudoku Game and Solver** built using C++.
This project allows users to **play Sudoku interactively** and also provides an option to **automatically solve the puzzle** using a backtracking algorithm.

---

## 🚀 Overview

This application combines:

* 🎮 **User Gameplay** (manual Sudoku solving)
* 🤖 **Auto Solver** (backtracking algorithm)

It demonstrates strong fundamentals of **arrays, functions, and recursion**.

---

## ✨ Features

* 🎯 Interactive Sudoku gameplay
* ✅ Real-time move validation
* ❌ Invalid move detection
* 🚫 Prevent overwriting filled cells
* ❤️ Limited attempts (lives system)
* 🎉 Win detection
* 🤖 One-click auto-solver (backtracking)

---

## 🧠 Concepts Used

* 2D Arrays
* Functions & Modular Design
* Recursion (Backtracking)
* Conditional Logic
* Input Validation
---

## 🎮 How to Play

1. Enter **row (0–8)** and **column (0–8)**
2. Enter a number (**1–9**)
3. System checks if move is valid
4. Solve the puzzle or:

   * Press **1** → Auto solve
   * Press **2** → Continue playing
